## Running the app
*We are split two folder: client and server*

`Run only client`

```bash
# development

$ npm run start | yarn start

# build mode

$ npm run build | yarn build


```

`Run only server`

```bash
# development

$ npm run start:dev | yarn start:dev

# production mode

$ npm run start:prod | yarn start:prod


```

## Run SSR

*Run script in root folder*

```bash

cd {root project}.

# bash script

$ ./ssr.sh

```
